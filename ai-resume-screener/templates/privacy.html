Privacy

Last updated: 2026-04-27

What this app does

This app compares a resume PDF against a job description and returns a score, matching/missing skills, suggestions, and (when available) evidence.

What data is sent to the AI model

  • If the LLM is used, the app sends redacted resume text and job description text to Gemini.
  • Redaction removes common identifiers like emails, phone numbers, and URLs (replaced with placeholders like [EMAIL], [PHONE], [URL]).

What is stored

  • Your uploaded PDF is stored only as a temporary file during processing and is deleted after the job finishes.
  • The server keeps a short-lived in-memory job/result record for polling (expires automatically).

Bring Your Own (BYO) API key

  • You can paste your own Gemini API key in the UI.
  • The key is saved in your browser only (localStorage) and sent to the server per request using the X-Gemini-Api-Key header.
  • If the server is configured with a background queue (Redis/RQ), your request (including the key) may be held briefly while processing, then discarded.

Limits

  • No system can guarantee perfect accuracy. Always verify important details.
  • If Gemini rate-limits or is unavailable, the app returns a local fallback analysis.
